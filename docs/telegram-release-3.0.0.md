🎉 **Zapret Hub 3.0.0 наконец-то вышел!**

Это самое большое обновление проекта за всё время. Последние месяцы приложение практически собиралось заново: новый интерфейс, новые компоненты, новая логика обхода и полноценная экосистема модификаций.

🛒 **Главное нововведение — [Zapret Marketplace](https://goshkow.com/zapret-hub/marketplace).**

Модификации для Zapret и Zapret 2 теперь не нужно искать и устанавливать вручную. Их можно посмотреть, скачать, обновить и удалить прямо внутри Hub. У Marketplace также появился отдельный сайт, через который авторами можно делиться своими проектами.

🤖 **Появился автоматический режим обхода.**

Zapret Hub может самостоятельно анализировать соединение, подбирать рабочую конфигурацию и переключать стратегии. Ручной режим никуда не пропал — полный контроль всё ещё остаётся у пользователя.

✨ **Что ещё появилось:**

• поддержка Zapret 2;
• полностью новый интерфейс на React, HTML и CSS;
• обновлённые быстрый доступ, настройки и onboarding;
• новые темы, анимации и звуки интерфейса;
• управление загрузками Marketplace;
• новый slim-установщик;
• множество исправлений стабильности и логики компонентов.

🔒 Приложение по-прежнему открытое и не отправляет телеметрию. Весь исходный код доступен на GitHub.
